The project combines 3D printing, embedded programming, and robotics to create a functional 5-DOF robotic arm capable of pick-and-place tasks. The hardware consists of an Arduino-based control system, MG996R servo motors, a dedicated power supply and 3D-printed parts from an already existing project in Thingiverse by another person(credits to: https://www.thingiverse.com/thing:7069225). The software is developed in C/C++ using the Arduino framework and provides real-time control of the robotic arm's movements. The project serves as a practical demonstration of robotic manipulation, actuator control, and mechatronic system integration.
